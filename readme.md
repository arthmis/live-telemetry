# Telemetry Overlay Assetto Corsa Evo

I made this for fun and because I wanted to have an overlay to see my pedal inputs as I'm driving. It's been pretty helpful so far. You can download an installable binary under releases or build it yourself. You'll need rust. You can find [its tooling here](https://rust-lang.org/) and how to get started. Try it out and let me know if there are any bugs. This only works on windows!
